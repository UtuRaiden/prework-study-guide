HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The li command is what adds a new bullet or number in a list.
  • The ul command needs to encompass all li's and it will creat a bulleted list.
  • The start of a section is inside the arrowheads and the end of one is inside arrowheads with a / at the front.
  • HTML is the framework of websites.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can use either a color name or Hex code to chose a color
  • CSS is the colors and sizes of the website and how it visibly looks

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • cd allows you to enter a directory inside of the one you already are in.
  • pwd is Print working Directory. Shows what directory you are in currently
  • git add -A : stages any changed files.
  • cd .. move up one directory
  • git checkout = move to branch
  • git checkout -b = create and move to new branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • All lines must end in a ;
  • Javascript can change a variable class after it is defined
  • JavaScript is the "Active" aspect of software. The buttons and other things
